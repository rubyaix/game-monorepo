# Micro Drift 게임 디자인 스펙

## 1. 트랙
- 형태: 오벌(직선 2 + 코너 2)
- 주요 파라미터
  - `straightHalf = 90`
  - `turnRadius = 60`
  - `roadHalfWidth = 18`
  - `wallHalfWidth = 28`
  - `START_S = 20`

## 2. 카트 이동 모델
- 목표 속도 기반(가속 개념 단순화)
- 오프로드/부스트/스턴 상태에 따라 목표 속도 변조
- 드리프트 시 조향 민감도 + 측면 슬립 적용

## 3. 드리프트 밸런스
- 드리프트는 키 홀드로 충전, 키 해제 시 부스트 발동
- 짧은 충전은 리턴 없음
- 긴 충전일수록 직선 탈출 가속 우위 확보

## 4. 충돌/리스크
- 벽 초과 시 트랙 가장자리로 클램프
- 진행 방향 180도 반전 및 스턴(`0.3s`)
- 스턴 중 속도 0, 드리프트/부스트 취소

## 5. 고스트
- 러닝 프레임(`t, x, y, z, heading`) 기록
- 세션 베스트 총시간 갱신 시 고스트 덮어쓰기
- 토글 비활성 시 렌더 숨김

## 6. 카메라
- 플레이어 후방 추적(lerp)
- 진행 방향 선행 지점 바라보기
- 고속에서도 트랙 가시성 확보 우선
